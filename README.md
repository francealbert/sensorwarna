Penjelasan:
Program ini mengatur sensor TCS3200 untuk membaca frekuensi warna merah, hijau, dan biru secara bergantian.
Kode pulseIn(sensorOut, LOW) digunakan untuk mengukur durasi pulsa rendah dari pin sensor, yang berkaitan dengan intensitas warna tertentu.
Hasilnya ditampilkan pada Serial Monitor untuk setiap warna yang terdeteksi.
Penggunaan:
Hubungkan sensor warna sesuai skema koneksi.
Unggah kode ke Arduino.
Buka Serial Monitor di Arduino IDE untuk melihat hasil deteksi warna.
